# POWER_BI-spotifyproject


## üìä Project Overview
This project analyzes personal Spotify listening data to uncover patterns, preferences, and trends over time. Using Power BI for advanced data visualization and DAX for complex calculations, the dashboard transforms raw streaming history into interactive insights across albums, artists, and tracks.

## üéØ Key Features
- **Three-Page Interactive Dashboard:** Overview, Listening Patterns, and Details pages
- **Dynamic Visualizations:** Matrix charts, scatter plots, and heatmaps with interactive elements
- **User-Driven Parameters:** Customizable visualizations including gradient quadrants in scatter plots
- **Time-Based Analysis:** Year-over-year and monthly listening trends from 2015 to present
- **Drillthrough Capability:** Click-through functionality from overview to detailed album information

## üõ† Technical Skills Demonstrated
- **Power BI & DAX:** Complex formulas, calculated columns, measures, and FORMAT function mastery
- **Data Visualization:** Heatmaps via matrix charts, parameter-driven scatter plot quadrants
- **Data Modeling:** Dataset relationships enabling seamless filtering and drillthrough
- **Data Analysis:** Listening frequency, duration, and content performance analysis

## üîç Key Insights Discovered

### Listening Behavior
- **Peak Hours:** Maximum activity between 8 PM - 11 PM and 12 AM - 3 AM
- **Session Patterns:** Mix of short sessions (0-5 min) and extended listening periods
- **Weekly Trends:** Consistent listening across all days with evening peaks

### Content Preferences
- **Top Artist:** The Beatles dominating at 12.96% of total plays
- **Album Variety:** Diverse library with 402 unique albums in analysis period
- **Track Engagement:** "Ode To The..." as most played track with significant lead

### Temporal Trends
- **Yearly Growth:** Notable increase in 2024 across all metrics
- **Consistent Engagement:** Sustained listening habits over multiple years
- **Seasonal Patterns:** Identifiable monthly variations in listening intensity
